# DramaBox React App

A premium dark-themed React application inspired by DramaBox, built with Vite and Tailwind CSS v4.

## Features

- **Premium UI**: Sleek dark mode design with glassmorphism effects and smooth transitions.
- **Drama List**: Browse a curated list of trending dramas with high-quality images.
- **Search**: Real-time search functionality to find dramas by title.
- **Pagination**: Navigate through pages of content.
- **Responsive Navbar**: Modern navigation bar with streamlined access to categories.

## Tech Stack

- **Framework**: React 19 (via Vite)
- **Styling**: Tailwind CSS v4 (using `@tailwindcss/postcss`)
- **Icons**: Lucide React

## Setup & Running

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser to the URL shown in the terminal (usually `http://localhost:5173` or similar).

## Notes

- This project uses **Tailwind CSS v4**. Configuration is handled via `postcss.config.js` and `@import "tailwindcss"` in `index.css`.
